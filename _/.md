Public Lookup API
/api/v1/lookup
    GET    /{code}                   (lang, date) <-> DictionaryLookupResponse
    GET    /{code}/{key}             (date) <-> ItemLookupResponse
    POST   /batch                    LookupBatchRequest <-> List<DictionaryLookupResponse>

Admin API: Definitions
/api/v1/admin/definitions
    GET    /                         (page, size, search) <-> PagedDefinitionResponse<DefinitionResponse>
    POST   /                         DefinitionCreateRequest <-> DefinitionResponse
    GET    /{code}                   (пустое тело) <-> DefinitionResponse
    GET    /{code}/versions          (пустое тело) <-> List<DefinitionResponse>
    POST   /{code}/versions          DefinitionCreateRequest <-> DefinitionResponse
    PATCH  /{code}/versions/{version}/publish  (пустое тело)<-> DefinitionResponse

Admin API: Items
/api/v1/admin/definitions/{code}/items
    GET    /                         (q, page, size, sort, status) <-> PagedDefinitionResponse<ReferenceItemResponse>
    GET    /{key}                    (пустое тело) <-> ReferenceItemResponse
    PUT    /{key}                    ItemUpsertRequest <-> ReferenceItemResponse
    POST   /batch                    List<ItemUpsertRequest> <-> BatchResponse
    DELETE /{key}                    (пустое тело) <-> (пустое тело)
    POST   /{key}/approve            (пустое тело)<-> ReferenceItemResponse

System API
/api/v1/admin/system
    DELETE /cache/{code}            (пустое тело) <-> (пустое тело)
    POST   /validate/{code}         (пустое тело) <-> ValidationReportResponse
    GET    /export/{code} (format=xlsx | json) (пустое тело) <-> byte[] | json
